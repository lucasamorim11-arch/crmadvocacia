1 Sistema de Login e Autenticação

Segurança JWT com controle de acesso por perfil de usuário.

🔐

Tela de Login

Acesso protegido por e-mail e senha. Sem login não é possível acessar nenhuma parte do sistema.

🪙

Token JWT

Ao entrar, um token seguro é gerado e salvo no navegador. Todas as chamadas à API são autenticadas automaticamente.

👤

Perfis de Acesso

Admin — acesso total, incluindo gestão de usuários.
Advogado — acesso às funcionalidades do dia a dia.

🚪

Nome + Botão Sair

O nome do usuário logado aparece no canto superior direito e na barra lateral, com botão para sair com segurança.

2 Área de Chat — WhatsApp

Interface completa para visualizar e responder mensagens do WhatsApp diretamente no CRM.

📋

Lista de Conversas

Coluna esquerda com foto, nome do contato (ou número), última mensagem e horário. Busca por nome ou número.

💬

Área de Mensagens

Estilo WhatsApp: mensagens enviadas à direita em verde, recebidas à esquerda em cinza. Com horário de cada mensagem.

⚡

Tempo Real

O chat atualiza automaticamente a cada 3 segundos. Novas mensagens recebidas via webhook aparecem sem precisar recarregar a página.

🔗

Vínculo com Lead

Quando uma mensagem chega, o sistema identifica automaticamente o lead pelo número de telefone e exibe o nome no lugar do número.

Como funciona o recebimento de mensagens?

Quando alguém envia uma mensagem pelo WhatsApp, a Evolution API repassa para o CRM via webhook (/webhook/evolution). O sistema salva a mensagem, vincula ao lead (se existir) e ela aparece no chat em até 3 segundos.

3 Conexão com WhatsApp (Evolution API)

Configuração da integração WhatsApp em Configurações → WhatsApp.

⚙️

Configuração Técnica

Campos para URL da Evolution API, API Key e nome da instância. Configuração salva no banco de dados.

📱

QR Code

Botão "Conectar WhatsApp" que cria a instância e exibe o QR Code para escanear com o celular.

🔄

Status Automático

O status da conexão (Desconectado / Conectado) atualiza sozinho a cada 5 segundos. Ao conectar, o QR Code some.

🔌

Desconectar

Botão para desconectar o WhatsApp a qualquer momento, sem perder o histórico de mensagens.

📲 Como conectar o WhatsApp

  1. Acesse Configurações no menu lateral
  2. Preencha a URL da Evolution API, API Key e nome da instância
  3. Clique em "Conectar WhatsApp"
  4. Abra o WhatsApp no celular e vá em Dispositivos conectados
  5. Toque em Conectar dispositivo e escaneie o QR Code
  6. O status muda automaticamente para Conectado ✅
+ Demais Funcionalidades do CRM

O sistema inclui um conjunto completo de ferramentas para gestão comercial.

📊

Dashboard

KPIs em tempo real: leads do mês, propostas, taxa de conversão, receita estimada e metas mensais com gráfico.

🗂️

Funil de Vendas (Kanban)

Quadro Kanban com 6 etapas. Arraste os cards entre colunas para mover leads pelo funil. Registro de motivo de perda.

👥

Leads & Clientes

Cadastro completo com CPF/CNPJ, telefone, WhatsApp, origem, área do direito, observações e histórico.

📄

Propostas

Criação e envio de propostas de honorários, controle de status (rascunho / enviada / aceita / recusada).

🔔

Follow-ups

Agendamento de próximos contatos com alertas de tarefas vencidas. Histórico completo por lead.

📢

Envio em Massa

Disparo de mensagens WhatsApp para múltiplos leads com filtros por origem e status. Templates personalizáveis.

📈

Relatórios

Relatórios por período com receita por origem, área do direito e quantidade de leads. Exportação visual.

🛡️

Gestão de Usuários

Cadastro de advogados e admins, ativação/desativação de contas. Acesso exclusivo para administradores.
